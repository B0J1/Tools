# How to use?

## Incloud what?
- .vimrc
- .tmux.conf
- .tmux.conf.local
- iTerm2_Profile.itermcolors

## Step 1
- ```$ brew install vim```
- ```$ brew install tmux```

## Step 2
- ```$ git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim```

## Step 3
- ```$ mv .vimrc ~```
- ```$ mv .tmux.conf ~```

## Step 4
- ```$ vim```
- ```$ PluginInstall```
